# Scalar-hiding Delegated MSM

---

## Prerequisites

* [NTL](https://libntl.org)
* [BLST](https://github.com/supranational/blst/tree/master)
* GMP
* OpenSSL

Modify the build script depending on your installation.

---

## Configuration

You can modify some basic parameters:

* $n$: The size of the MSM basis
* $\kappa$: The LPN dimension.
* Noise Rate: Controls sparsity of the error vector $e$.
